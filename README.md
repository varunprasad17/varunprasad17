### 📦 [xlsx-streamer](https://github.com/varunprasad17?tab=repositories) · *Published on PyPI*

A Python library that solves **Out-of-Memory crashes** when processing large Excel files.

Traditional tools (Pandas, Openpyxl) load entire files into RAM. `xlsx-streamer` **streams XML components on-the-fly**, cutting peak RAM usage by **90%+** — making multi-GB Excel processing viable inside Kubernetes Pods or AWS Lambda.

```bash
pip install xlsx-streamer[all]
```

</td>
<td width="40%" align="center">

```
📊 BEFORE  │  xlsx-streamer
───────────┼────────────────
  ~8 GB    │     ~200 MB
  RAM peak │   RAM peak
           │
  💥 OOM   │   ✅ Stable
  crashes  │   streaming
```
